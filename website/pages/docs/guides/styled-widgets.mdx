## Styled Widgets

These are widget primitives that can use _Mix_ `Attributes` and be rendered with those `Attributes`.

Initially, these widgets may seem complex. However, they primarily wrap basic _Flutter_ widgets and allow their visual properties to be defined through a _Mix._

The foundational widget primitive in _Mix_ is a `Box`. It can contain other widgets and a set of `BoxAttributes` to specify its visual properties.

> It's important to note that a `Box` is not like a `Container`, but rather, it is a `Container`. This means that the code:

That means that this:

```dart
Box(
  style: Style(
    height(100),
  ),
);
```

will translate to:

```dart
Container(height: 100);
```

Consider `Utilities` as shortcuts for defining visual properties. For instance:

```dart
Box(
  style: Style(
    border.radius(100),
  ),
);
```

will be rendered as:

```dart
Container(
  decoration: BoxDecoration(
    borderRadius: BorderRadius.circular(100),
  ),
)
```

## Catalog of Styled Widgets

Build-in styled widgets are:

### Box

`Box` is, basically, a `Container`. A widget that will usually be the base widget to create most of your layouts.

```dart
Box(
  style: Style(
    height(100),
    width(300),
    border.radius(15),
    backgroundColor.amber(),
    alignment.center(),
    elevation(4)
  ),
),
```

### FlexBox

A flex box is a widget that takes a list of children and displays them according to its `axis`.

```dart
FlexBox(
  style: Style(gap(6.0)),
  axis: Axis.horizontal,
  children: [
    IconMix(icon: Icons.search),
    TextMix('Click to search'),
  ],
)
```

#### HBox and VBox

`HBox` and `VBox` are helper widgets that are, basically, a `FlexBox` under the hood. `axis` is, respectively, `Axis.horizontal` and `Axis.vertical`

```dart
Hbox(
  style: Style(gap(6),),
  children: [
    Image.fromAsset('assets/image.png'),
    VBox(
      children: [
        TextMix('Title', variant: title),
        TextMix("""...description...""", variant: paragraph),
      ],
    ),
  ],
),
```

### StyledIcon

It's capable of displaying icons and inherit data from parent mixes

```dart
StyledIcon(
  style: Style(
    iconSize(30),
    iconColor(Colors.grey),
  ),
  icon: Icons.add,
),
```

### Pressable

A `Pressable` is a styled widget that can provide the following information for its descendants:

- hovering
- pressing
- disabled
- focused

Here's how you can create a text button with it:

```dart
Pressable(
  onPressed: () => debugPrint('my awesome button was pressed'),
  child: Box(
    style: Style(
      // when nothing's happening, bg color is the default grey
      backgroundColor(Colors.grey),
      (hover)(
        // when hovering, bg color is a lighter grey
        backgroundColor(Colors.grey.shade300),
      ),
      (press)(
        // when pressing, bg color is a darker grey
        backgroundColor(Colors.grey.shade600),
      ),
      borderRadius(8),
      padding(10),
      alignment(Alignment.center),
    ),
    child: TextMix(
      'PRESS ME',
      style: Style(
        (disabled)(
          text.style(color: Colors.grey.shade900),
        ),
        text.style(color: Colors.black, fontSize: 18.0),
        bold(),
      ),
    ),
  ),
),
```

### StyledText

It's capable of rendering text and inherit data from parent mixes

```dart
StyledText(
  'Mix is AWESOME!'
  style: Style(
    text.style(
      color: Colors.black,
      fontSize: 16.0,
      shadow: Shadow(
        color: Colors.black.withOpacity(0.2),
        offset: const Offset(2.0, 2.0),
        blurRadius: 1,
      ),
    ),
    bold(),
  ),
),
```
